# PymordialBlue User Guide

PymordialBlue is a robust automation framework designed for controlling Android emulators (specifically BlueStacks) via ADB and computer vision. It provides a structured way to manage device connections, application lifecycles, and UI interactions using visual recognition and OCR.

## Documentation Sections

*   **[Configuration](configuration.md)**: Setup instructions and configuration management.
*   **[Core Concepts](core_concepts.md)**: Understanding the Conroller and Application model.
*   **[Extract Strategies](extract_strategies.md)**: Customizing OCR preprocessing pipelines.

### Devices
*   **[ADB Device](devices/adb.md)**: Low-level Android Debug Bridge communication.
*   **[BlueStacks Device](devices/bluestacks.md)**: Emulator process management.
*   **[UI Device](devices/ui.md)**: Computer vision and template matching.
*   **[Tesseract Device](devices/tesseract.md)**: Optical Character Recognition (OCR).
